# Vanado kvalifikacijski zadatak

## Upute za pokretanje projekta

### 1. Preduvjeti
- .NET 10 SDK
- PostgreSQL
- Visual Studio 2022 ili noviji
- Entity Framework Core Tools

### 2. Postavljanje baze podataka
- Pokrenuti PostgreSQL server.
- Kreirati bazu podataka imenom **VanadoDB**
- U datoteci _appsettings.json_ provjeriti te po portrebi promijeniti connection string
- Pokretanje migracija naredbom _Update-Database_ sa NuGet Package Managerom

### 3. Pokretanje aplikacije
- Kliknuti Start ili F5 u Visual Studiu
- Otvoriti aplikaciju na _https://localhost:7131_
